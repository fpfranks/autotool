What is n8n? Free automation tool (like Zapier but self-hosted and unlimited). Install free at n8n.io or run locally. Import these JSON files to get all automations working.
Workflow 1: Lead Scraper + Email Sender

Runs daily. Reads leads from a Google Sheet, calls Claude API to personalise each email, sends via Gmail. Marks sent in sheet.

TriggerDaily schedule (9am UK time)
InputGoogle Sheet with leads
AIClaude personalises each email
Sends viaGmail (your account)
Limit20 emails/day (safe for deliverability)
{
  "name": "UK Lead Outreach — Daily Email Sender",
  "nodes": [
    {
      "type": "n8n-nodes-base.scheduleTrigger",
      "name": "Daily 9am UK",
      "parameters": {
        "rule": { "interval": [{"field":"cronExpression","expression":"0 9 * * 1-5"}] }
      }
    },
    {
      "type": "n8n-nodes-base.googleSheets",
      "name": "Get Unsent Leads",
      "parameters": {
        "operation": "readRows",
        "sheetId": "YOUR_SHEET_ID",
        "range": "Leads!A:G",
        "filterRows": true
      }
    },
    {
      "type": "n8n-nodes-base.splitInBatches",
      "name": "Process 20/day",
      "parameters": { "batchSize": 20 }
    },
    {
      "type": "n8n-nodes-base.httpRequest",
      "name": "Claude — Write Email",
      "parameters": {
        "method": "POST",
        "url": "https://api.anthropic.com/v1/messages",
        "headers": {
          "x-api-key": "YOUR_CLAUDE_API_KEY",
          "anthropic-version": "2023-06-01"
        },
        "body": {
          "model": "claude-haiku-4-5-20251001",
          "max_tokens": 500,
          "messages": [{
            "role": "user",
            "content": "Write a short UK cold email to {{$json.business_name}}, a {{$json.niche}} in {{$json.city}}. Offer our AI automation service. Include subject line. 4 sentences max. Casual British tone. CTA: visit [YOUR_SALES_PAGE_URL]. Sign off as [YOUR_NAME]."
          }]
        }
      }
    },
    {
      "type": "n8n-nodes-base.gmail",
      "name": "Send Email",
      "parameters": {
        "to": "={{$json.email}}",
        "subject": "={{$json.claude_subject}}",
        "message": "={{$json.claude_body}}"
      }
    },
    {
      "type": "n8n-nodes-base.googleSheets",
      "name": "Mark as Sent",
      "parameters": {
        "operation": "updateRow",
        "sheetId": "YOUR_SHEET_ID",
        "values": { "status": "emailed", "sent_date": "={{$now}}" }
      }
    }
  ]
}
Workflow 2: Follow-Up Sequence

Checks daily for leads emailed 3, 7, or 14 days ago with no reply. Sends appropriate follow-up. Stops automatically on reply.

TriggerDaily schedule
ChecksDays since first email
Sends3 follow-ups then stops
Auto-stopsWhen reply detected
{
  "name": "UK Follow-Up Sequence — Days 3/7/14",
  "nodes": [
    {
      "type": "n8n-nodes-base.scheduleTrigger",
      "name": "Daily 10am",
      "parameters": {
        "rule": { "interval": [{"field":"cronExpression","expression":"0 10 * * 1-5"}] }
      }
    },
    {
      "type": "n8n-nodes-base.googleSheets",
      "name": "Get Emailed No Reply",
      "parameters": {
        "operation": "readRows",
        "filter": "status = emailed AND replied = false"
      }
    },
    {
      "type": "n8n-nodes-base.if",
      "name": "Check Days Since Email",
      "parameters": {
        "conditions": [
          {"value1":"={{$json.days_since_email}}","operation":"equals","value2":3},
          {"value1":"={{$json.days_since_email}}","operation":"equals","value2":7},
          {"value1":"={{$json.days_since_email}}","operation":"equals","value2":14}
        ]
      }
    },
    {
      "type": "n8n-nodes-base.httpRequest",
      "name": "Claude — Write Follow-up",
      "parameters": {
        "body": {
          "messages": [{
            "role": "user",
            "content": "Write follow-up #{{$json.followup_number}} for {{$json.business_name}}. Day {{$json.days_since_email}} follow-up. Keep very short. British casual. Last one (day 14) is a 'closing the loop' email."
          }]
        }
      }
    },
    {
      "type": "n8n-nodes-base.gmail",
      "name": "Send Follow-up",
      "parameters": {
        "to": "={{$json.email}}",
        "subject": "Re: ={{$json.original_subject}}"
      }
    }
  ]
}

Workflow 5: Auto-Deploy Client Automations (Claude Agent)
This is the "zero touch" workflow. When a client pays and fills your Tally onboarding form, this workflow fires automatically. Claude reads their business details, generates a fully customised n8n workflow JSON for them, then uses the n8n API to create and activate it — all without you doing anything.

What gets auto-deployed: their AI reply bot, email templates, lead tracking sheet, and monthly report scheduler — all pre-filled with their business name, niche, and location.

What the client does once (5 mins): Click "Connect Gmail" and "Connect Google Business" — OAuth logins you legally cannot do on their behalf.

Workflow 6: AI Sales Responder (Claude closes for you)
This handles every reply to your cold emails — without you. When a prospect replies, n8n intercepts the email, Claude reads the full context (your original email + their reply), then writes and sends a response. Claude handles "how does it work?", "how much?", "sounds interesting tell me more", and even pushes back on objections — all automatically in British English. The goal of every reply is to get them to visit your sales page and buy.
TriggerNew reply in Gmail (label: outreach-replies)
Claude readsFull email thread + prospect's business type
Claude writesPersonalised reply in British English
HandlesInterest, questions, objections, pricing, "not now"
Always ends withLink to your sales page
You seeA copy of every AI reply (Bcc to your email)
