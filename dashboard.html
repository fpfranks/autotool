How this works
1 — NEW
Customer pays on Stripe. Click Send Welcome Email — Gmail opens pre-filled. Hit send.
2 — WELCOME SENT
Waiting for their ✅ Done reply. If no reply in 24hrs click Send Nudge. When they reply, click Mark Setup Done.
3 — SETUP DONE
Their access is ready. Build their ManyChat flows (~20 mins). Then click Send Live Email.
4 — BOT LIVE
All done. Client is active, bot is running 24/7. Nothing more needed unless they want updates.

Setup guide
  1. Go to script.google.com and click New Project
  2. Delete the default code and paste the contents of assets/apps-script.gs
  3. Click Deploy → New deployment
  4. Type: Web app — Execute as: Me — Who has access: Anyone
  5. Click Deploy, copy the URL, paste it above
