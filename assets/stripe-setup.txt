STEP 6 — SET UP WEBHOOK FOR N8N
========================================================

Your n8n automation needs to know when someone has paid so it can trigger the onboarding email automatically.

Step-by-step:

1. In Stripe Dashboard → Developers → Webhooks → + Add endpoint

2. Endpoint URL format:
   https://[YOUR-N8N-INSTANCE]/webhook/stripe-payment

   If using n8n Cloud: your URL will look like:
   https://[your-workspace].app.n8n.cloud/webhook/stripe-payment

   If self-hosted: use your server URL with the n8n webhook path

3. Events to listen for (select these):
   - checkout.session.completed
   - payment_intent.succeeded
   - customer.subscription.created
   - invoice.payment_succeeded

4. Click "Add endpoint" → Stripe generates a Signing Secret (starts with "whsec_")

5. Copy the Signing Secret immediately — you won't see it again unless you reveal it

6. In your n8n workflow:
   - Add a "Stripe Trigger" node (or use a Webhook node + manual Stripe verification)
   - Paste the Signing Secret into the Stripe node credentials
   - Set the webhook URL to match what you entered in Stripe

7. Test the webhook:
   - In Stripe → Webhooks → [your endpoint] → "Send test webhook"
   - Event type: checkout.session.completed
   - Verify the n8n workflow triggers correctly

WHAT YOUR N8N WORKFLOW SHOULD DO ON TRIGGER:
- Extract customer name, email, and phone from the Stripe payload
- Send a confirmation email with the Tally onboarding form link
- Add a row to your Google Sheet (leads tracker)
- Optional: Send yourself a WhatsApp notification via Twilio or Meta API

========================================================
STEP 8 — ONGOING STRIPE MANAGEMENT
========================================================

Cancellations:
- If a client wants to cancel: Dashboard → Customers → find their name → Subscriptions → Cancel
- This stops future payments. You can choose whether to cancel immediately or at the end of the period.

Refunds:
- Dashboard → Payments → find the payment → Refund
- For partial refund (e.g. keeping setup fee, refunding first month): enter the amount manually
- Stripe returns its processing fee for refunds within 90 days in most cases

Revenue tracking:
- Dashboard → Reports → Revenue — gives you monthly MRR (Monthly Recurring Revenue) view
- Export as CSV monthly for your accounts
